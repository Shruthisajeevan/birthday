# 🤍 Birthday Wish — for my love

A deeply personal, cinematic birthday surprise page.
Opens with a flickering candle → tap → confetti + floating petals + scrollable love letter.

---

## 📸 How to Add Your Photo

1. Put your favourite photo in this folder, name it `photo.jpg`
2. Open `index.html`, find `pol-placeholder` and replace:
```html
<div class="pol-placeholder">...</div>
```
with:
```html
<img src="photo.jpg" alt="us"/>
```
